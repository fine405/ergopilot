# Teaching notes

- Teaching language: Chinese, with English technical terms retained where useful.
- Starting point: professional software engineer with frontend emphasis.
- Reuse familiar React, state-management and HTTP concepts as bridges.
- Do not assume prior Rust, distributed-systems, IoT or robotics knowledge.
- Prefer short lessons, concrete ErgoPilot examples and immediate feedback.
- Separate concepts the learner must master from technologies they only need to
  recognize initially.
