# Mission: 从前端工程师成长为 Agent Runtime 工程师

## Why
为了胜任清闲智能一类软硬件结合的 Agent 工程师岗位，完成并能清楚讲解
ErgoPilot：一个可恢复、可观测、受权限约束的智能工位 Agent 项目。

## Success looks like
- 能用自己的话解释 Agent、工具、工作流、状态机、MCP、幂等和恢复
- 能实现自然语言到安全设备动作的端到端纵向切片
- 能区分模型错误、编排错误、传输错误和设备错误并系统定位
- 能说明每项技术的必要性、替代方案及取舍
- 能用测试、评测和真实指标支撑简历描述

## Constraints
- 已有软件工程经验，技术背景偏 TypeScript、React 和前端
- Rust、分布式可靠性和机器人/IoT 属于新领域
- 采用边做项目边学习的方式，作品集目标周期约 4–6 周
- 优先掌握最小可运行系统，不追求一次学全所有理论

## Out of scope
- 现阶段深入嵌入式驱动、电机控制和实时操作系统
- 现阶段深入 ROS 2、DDS、MoveIt2 和复杂动力学仿真
- 与项目无关的多 Agent、RAG、模型训练和 Kubernetes 知识
